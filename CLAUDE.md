## Repository Overview

This is the **public website** repository for **skelin.io** — a programming community in Iceland. Internal notes live in the separate private `skelin-hq/notes` repo.

## Repository Structure

- `site/` — Public website deployed to GitHub Pages (skelin.io)
  - `CNAME` — Custom domain configuration
  - `index.html`, `circles.html`, `manifesto.html`, `style.css`

## Contribution Workflow

Changes go through GitHub pull requests on the `main` branch. There are no CI checks or automated validations.

## Writing Conventions

- All content is written in Markdown
- The community uses a holacracy model — the circle-based organizational structure is described on the website (`site/circles.html`)
