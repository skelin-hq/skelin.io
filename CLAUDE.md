# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Overview

This is a **documentation-only repository** for **localhost.is** — a programming community in Iceland. There is no code, no build system, and no tests. All content is Markdown files.

## Repository Structure

- `README.md` — Community overview and values
- `MANIFESTO.md` — Core beliefs and organizational philosophy
- `CIRCLES.md` — Holacracy governance model and how to create circles
- `projects/` — Active community projects (e.g., `agent-innlit/` webinar series)
- `retros/` — Event retrospectives, named by date (`YYYY-MM-DD.md`)

## Contribution Workflow

Changes go through GitHub pull requests on the `main` branch. There are no CI checks or automated validations.

## Writing Conventions

- All content is written in Markdown
- Retrospective files are named `retros/YYYY-MM-DD.md`
- The community uses a holacracy model — respect the circle-based organizational structure described in `CIRCLES.md` when proposing governance changes
